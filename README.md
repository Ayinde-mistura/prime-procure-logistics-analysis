# PrimeProcure Logistics & Supply Ltd — Data Analysis Project

## Project Overview
This project presents a comprehensive data analysis of **1,000 shipment records**
across **six Nigerian cities** for PrimeProcure Logistics & Supply Ltd.
The analysis covers operational performance, cost efficiency, delivery tracking,
customer payment behaviour, and priority shipment management.

**Tools Used:** MySQL · Power BI · Microsoft Excel · Python (Streamlit)

## Business Questions Answered
- Which cities and routes have the highest delivery delays?
- What are the total and average logistics costs by shipment type?
- How does payment method affect revenue collection?
- Which shipment priorities generate the most revenue?
- What is the on-time delivery rate across all cities?

## Key Insights
- 📦 Over 1,000 shipments analysed across Lagos, Abuja, Kano, Port Harcourt, Ibadan, and Enugu
- 🚚 Identified top-performing and underperforming delivery routes
- 💰 Cost analysis revealed shipment categories with highest logistics spend
- 📊 Power BI dashboard built with DAX measures, custom DateTable, and star schema model
- 💳 Payment behaviour analysis across cash, transfer, and card channels
